## ✨ Engenharia de Interface e UX

Desenvolvi funcionalidades focadas em performance e otimização de recursos:

* **🌐 Sistema de Partículas:** Renderização de uma rede interativa (efeito constelação) via **API de Canvas**, processada em tempo real.
* **🎯 Movimento Suave (Lerp):** Cursor personalizado que utiliza **interpolação linear** para suavizar o rastreio do mouse.
* **⚡ Performance de Scroll:** Uso de **IntersectionObserver** para disparar animações e barras de progresso apenas quando os elementos estão visíveis.
* **⌨️ Micro-interações:** Efeito *typewriter* dinâmico, animações de *glitch* nos cards e um *Easter Egg* (Konami Code).
