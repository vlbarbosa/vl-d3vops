## ✨ Funcionalidades

- Cursor personalizado com efeito lerp suavizado
- Efeito typewriter com múltiplas frases
- Partículas interativas em canvas (constelação)
- Animações reveal on scroll (IntersectionObserver)
- Barras de habilidade animadas
- Contadores numéricos animados
- Parallax no hero
- Barra de progresso de leitura
- Micro-animação glitch nos cards
- Easter egg Konami Code (↑↑↓↓←→←→BA)
- 100% responsivo para mobile
